# Zama Protocol Quiz

**Put your Zama knowledge to the test.**

An interactive quiz covering the full Zama tech stack: fhEVM, TKMS, Coprocessors, Gateway, and TFHE-rs.

## What's Inside

- **31 questions** across 5 sections
- Topics: Architecture, Data Flow, Decryption, Service Interactions, Advanced FHE Concepts
- Progress saving (survives browser refresh)
- Review mode to learn from mistakes
- Mobile-friendly

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Build

```bash
npm run build
```

Static files output to `dist/` - deploy anywhere.

## Tech Stack

React + TypeScript + Vite + Tailwind CSS
